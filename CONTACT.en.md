## Types of collaboration

| Type | Description | First step |
|---|---|---|
| **Code contribution** | Bugfix, new feature, new extractor rule | Read [CONTRIBUTING.en.md](CONTRIBUTING.en.md), then open a Pull Request |
| **Data source** | New industry catalog, regional index, IHK data | Open an issue with source description and privacy notice |
| **Methodology** | Proposal to change weights or rules | Open an issue with scientific rationale and references |
| **Partnership** | Co-development, funded project, institutional collaboration | [GitHub Discussion](https://github.com/syrokomskyi/hdri/discussions) or e-mail via GitHub profile |
| **Press / Research** | Interview, citation, using the data for a study | Cite as: *Handwerk Digital Readiness Index (HDRI), github.com/syrokomskyi/hdri* |

## Response times

- GitHub Issues: typically within 7 days
- Pull Requests: review within 14 days
- Partnership enquiries: first response within 21 days

Replies are provided in English or German.
